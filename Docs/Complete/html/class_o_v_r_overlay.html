Detailed Description

Add OVROverlay script to an object with a Quad mesh filter to have the quad rendered as a TimeWarp overlay instead by drawing it into the eye buffer.

This will take full advantage of the display resolution and avoid double resampling of the texture.

If the texture is dynamically generated, as for an interactive GUI or animation, it must be explicitly triple buffered to avoid flickering when it is referenced asynchronously by TimeWarp.
